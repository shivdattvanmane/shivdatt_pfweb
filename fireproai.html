01 / AI Surveillance

FirePro AI

An intelligent real-time fire, smoke, and hazard detection system built with Python, Flask, OpenCV, YOLOv8, Firebase, and Socket.IO for smart surveillance workflows.

View Report View Screens

Project Concept

FirePro AI continuously monitors live camera feeds and identifies hazardous events with confidence scores, severity levels, timestamps, captured screenshots, email alerts, event history, Excel exports, and configurable detection settings.

Tech Stack

  • Python and Flask backend
  • OpenCV video processing
  • YOLOv8 model inference
  • Firebase Firestore storage
  • Socket.IO real-time updates
  • OpenPyXL Excel reporting

Key Features

  • Real-time fire and smoke detection
  • Human presence monitoring
  • Automated email notifications
  • Screenshot evidence capture
  • Detection history and event logs
  • Configurable alert cooldowns
